# 🌐 NEXUS WEB — Cinematic Multimedia Suite

The web-based counterpart to the **Nexus Studio** desktop suite. Built for the modern web with a focus on cinematic aesthetics, high-performance interactions, and a premium "Cyberpunk" design language.

---

## ⚡ Tech Stack
- **Framework**: React 18
- **Build Tool**: Vite (Lightning fast)
- **Animations**: Framer Motion (Liquid smoothness)
- **Icons**: Lucide React (Clean & Modern)
- **Styling**: Vanilla CSS (Maximum flexibility & control)

---

## ✨ Features
- **Glassmorphism UI**: Frosted blur panels and neon glow accents.
- **Cinematic Navigation**: Smooth page transitions and hover states.
- **Dashboard**: "Continue Watching" and "Quick Action" tiles.
- **Responsive Design**: Fluid layouts that adapt to your screen.

---

## 🛠 Getting Started

### 1. Install Dependencies
```bash
npm install
```

### 2. Run Development Server
```bash
npm run dev
```

### 3. Build for Production
```bash
npm run build
```

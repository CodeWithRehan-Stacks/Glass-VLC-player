**Nexus Studio** is a world-class multimedia platform available on both Desktop and Web. It combines the cinematic immersion of **Apple TV**, the content-rich experience of **Netflix**, and the professional-grade mixing power of **Adobe Audition** into a single, breathtaking AI-ready ecosystem.

---

## 🖥️ Desktop Suite (Nexus Studio)
A high-performance, native application built for the ultimate cinematic experience.

### 🌟 Key Features
-   **Cinematic Cinema Player**: Native `QtMultimedia` rendering with hardware acceleration and **Ambilight** glow.
-   **Live Audio Mixer**: Adobe Audition-inspired multi-channel mixing panel.
-   **Global Radio Hub**: Instant streaming of global stations (Lofi, Jazz, Synthwave).
-   **Smart Dashboard**: Netflix-style "Continue Watching" system with persistent memory.
-   **Glassmorphism UI**: Premium Cyberpunk design system (v3.0) with elastic animations.

### 📂 Desktop Structure
```text
/ultimate_suite
├── components/     # Custom UI widgets (TitleBar, Controls)
├── navigation/     # Animated Router & Elastic Sidebar
├── views/          # Dashboard & Screen layouts
├── mixer/          # Multi-channel audio engine
├── themes/         # Version 3.0 QSS Design System
└── main.py         # Application Orchestrator
```

## 🌐 Web Version (Nexus Web)
A lightning-fast, modern web counterpart designed for universal access.

### 🚀 Tech Stack
-   **Frontend**: React 18 + Vite
-   **Animations**: Framer Motion (Liquid smoothness)
-   **Styling**: Vanilla CSS (Premium Glassmorphism)
-   **Icons**: Lucide React

### 🌟 Key Features
-   **Seamless UI**: Mirrors the desktop's "Cyberpunk" aesthetic with high-end web effects.
-   **Animated Routing**: Fluid page transitions and hover interactions.
-   **Responsive Dashboard**: Adaptive layouts for any screen size.

---

## 🛠️ Installation & Setup

### For Desktop:
```powershell
cd ultimate_suite
pip install -r requirements.txt
python main.py
```

### For Web:
```powershell
cd nexus-web
npm install
npm run dev
```
